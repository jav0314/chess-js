## Setup
_[back to top](#table-of-contents)_

Clone the project

```
git clone git@github.com:jav0314/chess-js.git
```

Install the dependencies

```
bun install
```

## Useful commands

Build project on production mode

```
bun run build
```

Build project on developer mode (watch files)

```
bun run dev
```
